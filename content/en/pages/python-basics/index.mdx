The following links point to training material that might be useful if you are just getting started with python and machine learning.

**Basic Python free courses for beginners**

* Python website [The Python Tutorial](https://docs.python.org/3/tutorial/index.html)
* FreeCodeCamp "[Learn Python Basics – A Guide for Beginners](https://www.freecodecamp.org/news/learn-python-basics/)" by Chepkirui Dorothy
* W3Schools [Python Tutorial](https://www.w3schools.com/python/)

 

**Scientific Python**

**Books** (most of them have free access to exercises on GitHub)

* [Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow, 2nd Edition](https://www.oreilly.com/library/view/hands-on-machine-learning/9781492032632/) by Aurélien Géron
* [Deep Learning with Python](https://www.manning.com/books/deep-learning-with-python) by François Chollet
* [Deep Learning with PyTorch, Second Edition](https://www.manning.com/books/deep-learning-with-pytorch-second-edition) by Luca Antiga, Eli Stevens, Howard Huang, Thomas Viehmann
* [Deep Learning](https://www.deeplearningbook.org/) by Ian Goodfellow, Yoshua Bengio and Aaron Courville
